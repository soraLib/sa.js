Class Wrapper

description

create a Wrapper class to help you splice xml.

param attrs

attributes on xml root

param content

wrapped inner content

example

const xml = new Wrapper("some content").wrap('ctrl', { id: 20, name: "hfutsora"}) then you can get the xml <ctrl id="20" name="hfutsora">some content</ctrl>
